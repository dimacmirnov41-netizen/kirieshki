# 🥖 Russian Snacks: Red Caviar Rye Croutons (128x)

A high-quality, detailed resource pack for Minecraft that replaces the standard vanilla Cookie item with legendary Eastern European rye bread croutons. This edition features the most iconic and "fancy" flavor: Red Caviar.

---

## 📂 Repository Structure
* `ресурсы/майнкрафт` — Core texture files and localization assets.
* `pack.mcmeta` — Resource pack configuration and format specification.
* `pack.png` — Official project icon.

## 🛠️ Technical Info & Compatibility
This resource pack officially supports Minecraft versions from **1.20.1 up to 1.21.11**.
* *If you encounter any minor visual bugs or glitches, simply reload your textures in-game by pressing **F3 + T**.*
